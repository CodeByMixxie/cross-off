🍓 Cross Off — Aesthetic To-Do App

A strawberry-themed desktop To-Do application built with Python and CustomTkinter.This originally started as a beginner learning project, now evolved into a fully structured, class-based GUI application with storage and theme support.

✨ Features

Add tasks (Enter key supported)
Animated checkbox completion
strike effect
Light & Dark theme toggle (Strawberry & Berry Night modes)
Right-click task deletion
Storage using SQLite
Packaged as a standalone .exe
Refactored into class-based architecture

🛠 Tech Stack

Python 3
CustomTkinter
SQLite
PyInstaller
Git & GitHub

📌 Project Evolution

This project started as a simple beginner script and gradually evolved into a structured desktop application with:
